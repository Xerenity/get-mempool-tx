# Getting pending transactions from mempool via Ethers.

### Installation
`npm i`

Copy `websocket-provider.js` from extra folder to `/node_modules/@ethersproject/providers/lib/` and replace it.

Put your `rpcWS` in `.env`.

### Run
`node main`
